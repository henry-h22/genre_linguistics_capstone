# Phonetically-Motivated Feature Engineering for Genre Classification

Hey! This is just the research code for my capstone, dumping it here in case my computer explodes.

I'm trying my hand at music genre classification, using a dataset called [GTZAN](https://www.kaggle.com/datasets/andradaolteanu/gtzan-dataset-music-genre-classification/data), with measures inspired by phonetics. I'm using OpenAI's whisper model and [Montreal Forced Aligner](https://montreal-forced-aligner.readthedocs.io/en/latest/) and [PraatSauce](https://github.com/kirbyj/praatsauce) and Python and R. It didn't really work but it was interesting!
